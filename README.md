# Shufflr

![Shufflr](resources/card.svg)

Shuffle Spotify playlists such that consecutive songs are similar.

## Installation

```bash
pip install .
```

## Usage

```bash
python3 -m shufflr --clientSecret CLIENT_SECRET --inputPlaylists PLAYLIST1 PLAYLIST2
```
